Sobre mi

"Curso la universidad actualmente con el objetivo de obtener mi título de TSU y, en el futuro, continuar con una ingeniería. Creación de interfaces de usuario atractivas y fáciles de usar. Implementación de diseños utilizando HTML, CSS y JavaScript. Optimización de sitios web para una experiencia óptima del usuario.

Desarrollo Backend

Creación de servidores y bases de datos. Implementación de lógica empresarial y funcionalidades. Aseguramiento de la escalabilidad y eficiencia del sistema.
